# dots
Tim Davis' personal OSX dotfiles.

# Specifications:
- Operating System: [MacOS](https://apple.com/macos)
- Terminal Emulator: [Alacritty](https://github.com/alacritty/alacritty)
- Shell: [zsh](https://zsh.org)
- Editor: Mainly [Visual Studio Code](https://code.visualstudio.com), but [Neovim](https://neovim.io) for quick edits.

# Installation
1. Clone this repository:
```sh
$ git clone https://github.com/flmng0/dots
```
2. Change directory into cloned dots.
```sh
$ cd dots
```
3. Initialize submodules.
```sh
$ git submodule update --init --recursive
```
4. Link configurations.
```sh
$ ./link_configs.sh
```
